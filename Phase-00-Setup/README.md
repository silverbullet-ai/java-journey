# Phase 0 — Setup

## Objective

Set up the development environment required for the Java Fullstack learning journey.

---

## What Was Done

* Installed **Java (JDK 17/21 – LTS)**
* Verified Java installation using terminal
* Installed and configured **Git**
* Set up **IntelliJ IDEA**
* Created and ran the first Java program

---

## Tools Setup Overview

### Java (JDK)

Downloaded and installed JDK to compile and run Java programs.

### Git

Installed Git and configured username and email for version control.

### IntelliJ IDEA

Set up IntelliJ IDEA and created the first Java project.

---

## First Program

A basic Java program was created and executed to verify the setup:

```java
public class Main {
    public static void main(String[] args) {
        System.out.println("Java Fullstack Journey Begins!");
    }
}
```
