# Module 2: Object-Oriented Programming (OOP)

This module focuses on the core principles of Object-Oriented Programming in Java. It introduces how real-world entities are modeled using classes and objects, and how code can be structured for scalability and reuse.

---

## Topics Covered

- Class & Object  
- Constructors  
- Encapsulation (Access Modifiers, Getters & Setters)  
- Inheritance  
- Polymorphism (Overloading & Overriding)  
- Abstraction (Abstract Classes & Interfaces)

---

## Structure

Each topic is organized into:
- `notes.md` → Concept explanations  
- `README.md` → Topic summary  
- `src/` → Example and practice code  

---

## Key Learnings

- Designing classes to represent real-world entities  
- Controlling data access using encapsulation  
- Reusing code using inheritance  
- Writing flexible programs using polymorphism  
- Hiding implementation details using abstraction  

---

## Outcome

By completing this module, you should be able to:
- Design and implement object-oriented programs  
- Understand relationships between classes  
- Write clean, modular, and maintainable Java code
