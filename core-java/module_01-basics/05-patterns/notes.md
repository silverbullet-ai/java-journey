# Module 1 - Basics (Patterns & Nested Loops)

## Nested Loops

A loop inside another loop.

Syntax:

```java
for (int i = 1; i <= n; i++) {
    for (int j = 1; j <= m; j++) {
        // code
    }
}
```

---

## Key Concept

- Outer loop → controls rows
- Inner loop → controls columns

---

## Pattern Thinking

To solve patterns:
1. Identify rows
2. Identify columns
3. Decide what to print (number, star, space)

---

## Example Pattern

```
*
* *
* * *
* * * *
```

Logic:  

Row 1 → 1 star  
Row 2 → 2 stars  
...

---

## Important Points

- Always trace with pen & paper
- Focus on row-column relationship
- Start simple → increase complexity
