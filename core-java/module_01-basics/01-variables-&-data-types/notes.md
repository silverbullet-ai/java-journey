# Module 1 - Basics (Variables & Data Types)

## Variables
Variables are used to store data in Java.

Syntax:
datatype variableName = value;

Example:
int age = 21;

---

## Data Types

### 1. Primitive Data Types
- int → stores integers (e.g., 10)
- float → stores decimal numbers (requires 'f')
- double → stores larger decimal values
- char → stores a single character ('A')
- boolean → true or false

### 2. Non-Primitive Data Types
- String → sequence of characters (e.g., "Aahish")

---

## Important Concepts

### Strongly Typed Language
Java requires declaring the data type before using a variable.

### Float vs Double
- float → less precision (requires 'f')
- double → more precise (default for decimals)

Example:
float num = 5.5f;
double num2 = 5.5;

### String is Not Primitive
String is actually an object, not a primitive data type.

---

## Key Takeaways
- Always declare variable type
- Use meaningful variable names
- Understand difference between primitive and non-primitive
