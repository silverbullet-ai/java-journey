# Module 3 - Collections (Stack & Queue)

## Stack (LIFO)

Last In First Out

Example:

```java
Stack: [10, 20, 30]
Pop → removes 30
```
---

## Stack Methods

- push(value)
- pop()
- peek()

---

## Queue (FIFO)

First In First Out

Example:

```java
Queue: [10, 20, 30]
Remove → removes 10
```
---

## Queue Methods

- add(value)
- remove()
- peek()

---

## Important Concepts

- Stack → undo operations, recursion  
- Queue → scheduling, buffering  

---

## Key Takeaways

- Stack → LIFO  
- Queue → FIFO  
- Used in real-world systems
