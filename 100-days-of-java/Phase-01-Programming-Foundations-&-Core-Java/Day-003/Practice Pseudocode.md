## 🧠 Practice Questions

### 1. Sum of Two Numbers
```
START
INPUT A, B
SET Sum = A + B
PRINT Sum
END
```

### 2. Positive or Negative

```
START
INPUT N
IF N >= 0 THEN
    PRINT "Positive"
ELSE
    PRINT "Negative"
END IF
END
```

### 3. Largest of Two Numbers

```
START
INPUT A, B
IF A > B THEN
    PRINT "A is largest"
ELSE
    PRINT "B is largest"
END IF
END
```
