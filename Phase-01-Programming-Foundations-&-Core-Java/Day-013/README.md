## Objective
To understand method overriding in Java, its rules, use cases, and how it enables runtime polymorphism.

## Topics Covered

- Conditions and Rules
- @Override Annotation
- Overriding vs Overloading
- Polymorphism and Overriding
- super Keyword
- Limitations (final, static, private)
- Advantages of Method Overriding
